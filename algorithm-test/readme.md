# 알고리즘 & 자료구조

```bash
├── boj # 백준 알고리즘 문제 풀이 (문제번호-난이도)
├── pgs # 프로그래머스 알고리즘 문제 풀이(문제이름)
├── hacker rank # 해커랭크
└── leet # 리트코드
```

## Algorithm

- [BOJ] : 백준
- [PGS] : 프로그래머스
- [LEET] : 리드코드
- [Hacker]: 해커랭크

.
