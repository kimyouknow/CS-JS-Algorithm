# 문제 읽기

선수과목 조건

- 한 학기에 들을 수 있는 과목 수에 제한 없음
- 모든 과목은 매 학기 항상 개설

모든 과목에 대해 각 과목을 이수하려면 최소 몇 학기가 걸리는지 계산

1번 과목부터 N번 과목까지 차례대로 최소 몇 학기에 이수

과목의 수 N(1 ≤ N ≤ 1,000)과 선수 조건의 수 M(0 ≤ M ≤ 500,000)

A번 과목이 B번 과목의 선수과목이다. A < B인 입력만 주어진다. (1 ≤ A < B ≤ N)
